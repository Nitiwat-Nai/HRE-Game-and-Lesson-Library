🚀 เลือกเกมหรือบทเรียนที่ต้องการได้เลย

เพื่อประสบการณ์การเรียนรู้ที่สนุกสนานและมีประสิทธิภาพ! (คลิกที่ปุ่ม 'เข้าเล่น' ด้านล่างแต่ละหัวข้อ)

Word Icons Match Game

เกมจับคู่คำศัพท์กับไอคอนสนุก ๆ ฝึกทักษะการจำคำนามและกริยาพื้นฐาน

เข้าเล่นเกม

Irregular Verbs (กริยา 3 ช่อง)

บทเรียนและแบบฝึกหัดสำคัญสำหรับกริยาที่ไม่ปกติ (Irregular Verbs) ที่ใช้บ่อย

เข้าสู่บทเรียน

Word Search Game (ค้นหาคำศัพท์)

เกมค้นหาคำศัพท์สุดคลาสสิก เพื่อฝึกการสังเกตและจดจำตัวสะกด

เข้าเล่นเกม

WordWise Hangman

เกมแขวนคอในรูปแบบที่ออกแบบมาเพื่อฝึกการเดาคำศัพท์จากบริบท

เข้าเล่นเกม

Puzzle Word Trace

เกมต่อคำศัพท์แบบปริศนา ฝึกการจัดเรียงตัวอักษรให้เป็นคำที่ถูกต้อง

เข้าเล่นเกม

Alternatives to Very Challenge

บทเรียนท้าทายเพื่อเรียนรู้คำศัพท์ขั้นสูง แทนการใช้คำว่า "Very" ซ้ำ ๆ

เข้าสู่บทเรียน

Antonyms Quiz Challenge

แบบทดสอบท้าทายความรู้คำศัพท์ตรงข้าม (Antonyms) เพื่อเพิ่มคลังคำศัพท์

เข้าเล่นเกม

A-Z Alphabet Fun

บทเรียนและเกมสนุก ๆ สำหรับเด็ก เพื่อเรียนรู้ตัวอักษร A-Z และคำศัพท์พื้นฐาน

เข้าเล่นเกม

Homophones Game

เกมทดสอบความเข้าใจคำที่มีเสียงเหมือนแต่ความหมายและสะกดต่างกัน (Homophones)

เข้าเล่นเกม

Vocab for Kids (คำศัพท์เด็ก)

ชุดคำศัพท์ที่ออกแบบมาสำหรับเด็กโดยเฉพาะ พร้อมภาพประกอบน่ารัก

เข้าสู่บทเรียน

Word Journey

การเดินทางของคำศัพท์ ที่ให้ผู้เรียนค้นพบคำใหม่ ๆ ในแต่ละก้าว

เข้าสู่เกม

Vocab for Kids: Sentence Building

ฝึกการสร้างประโยคพื้นฐานสำหรับเด็ก โดยใช้คำศัพท์ที่เรียนรู้มา

เข้าสู่บทเรียน

Word Pop Match

เกมจับคู่คำศัพท์แบบป๊อปอัป ที่ต้องความเร็วและความแม่นยำ

เข้าเล่นเกม

Word Journey Prefix

การเรียนรู้คำศัพท์จากการทำความเข้าใจส่วนนำหน้า (Prefix) ของคำ
